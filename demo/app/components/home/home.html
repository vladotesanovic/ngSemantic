Getting Started

Angular2 building blocks based on Semantic UI

  • Angular 2 angular.io
  • Semantic UI semantic-ui.com
  • GitHub repository https://github.com/vladotesanovic/ngSemantic ( If you like it, give it a star )

Installation

In terminal: $ npm install ng-semantic --save

Semantic css and js ( with jQuery are required )

<link rel="stylesheet" type="text/css" href="https://cdnjs.cloudflare.com/ajax/libs/semantic-ui/2.2.2/semantic.min.css"> <script src="https://cdnjs.cloudflare.com/ajax/libs/jquery/3.1.0/jquery.min.js"></script> <script src="https://cdnjs.cloudflare.com/ajax/libs/semantic-ui/2.2.2/semantic.min.js"></script>

Angular CLI configuration


Use

// AppModule import { NgModule } from 'angular2/core'; import { NgSemanticModule } from 'ng-semantic'; @NgModule({ bootstrap: [ AppComponent ], declarations: [ AppComponent ], imports: [ BrowserModule, NgSemanticModule ] }) export class AppModule {} // Component import { Component } from 'angular2/core'; @Component({ selector: 'demo-cmp', template: '<sm-segment class="raised">Hello</sm-segment>' }) export class DemoComponent {}
